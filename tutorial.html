Tutorial do Labirinto Sequencial

Movimentos em sequência:

Use as setas do teclado para movimentar o personagem pelo labirinto obedecendo a seguinte sequência de cores: Azul, Vermelho e Amarelo.

Se pisar errado no azul você volta para o inicio.

Já o vermelho te tira um coração.

O amarelo não vai deixar você passar se não estiver na ordem certa.

Objetivos:

Pegue um Q para responder uma pergunta do Quiz! Se errar você perde um coração, mas você não é obrigado a responder todos, portanto use seu reciocíonio para responder tudo ou deixar alguns para trás.

As chaves vão liberando o caminho para você.

Se você pegar o troféu você ganha o jogo!
